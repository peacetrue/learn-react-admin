= 简介



一个前端框架，用于使用 ES6，React 和 Material Design 在 REST/GraphQL API 之上构建在浏览器中运行的管理应用程序。由 marmelab 开源和维护。

react-admin 基于 REST/GraphQL API 的设计，使其组件在调用 API 接口时非常方便，这是它与其他后台管理框架的一大区别。
